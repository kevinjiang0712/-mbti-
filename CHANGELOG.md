# Changelog

本文件记录猫咪MBTI人格测试小程序的所有版本变更。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.2] - 2026-03-23

### 新增

#### 分享功能
- **分享好友**：结果页新增「分享好友」按钮，支持微信好友分享
  - 分享卡片展示猫咪图片和人格类型
  - 自定义分享标题（如"我家猫是 INTJ：冷面霸总猫"）

#### 海报生成
- **保存海报**：Canvas 2D 绘制精美分享海报
  - 星空渐变背景 + 随机星星粒子
  - SSR 风格猫咪卡片（带发光边框）
  - 人格类型名称 + 性格标签
  - 小程序码（圆形白底）+ 引导文案
- **海报预览弹窗**：毛玻璃遮罩 + 悬浮动效
- **保存到相册**：支持权限引导，保存失败自动跳转设置页

#### 按钮布局
- 结果页底部采用双按钮并排布局：
  - 左侧「保存海报」（毛玻璃透明底）
  - 右侧「分享好友」（紫色渐变 + 发光）
- 「返回魔法入口」文字链接

#### 数据记录
- **测试结果云端存储**：每次测试完成后自动保存记录到云数据库
  - 记录 MBTI 类型、人格称号、原始得分、测试时间
  - 静默保存，不影响用户体验
  - 便于后续数据分析和用户画像

### 技术实现
- 云存储图片下载：`wx.cloud.downloadFile`
- Canvas 2D 离屏绘制 + `canvasToTempFilePath` 导出
- 并行加载猫咪图片和小程序码
- 相册权限处理 + `wx.openSetting` 引导
- 云数据库 `test_records` 集合存储测试记录

## [1.0.0] - 2026-03-22

### 新增

#### 核心功能
- 完整的MBTI人格测试流程：首页 → 测试页 → 结果页
- 12道精心设计的猫咪行为问题，覆盖四个维度：
  - E/I（外向/内向）：Q1, Q2, Q9
  - S/N（感知/直觉）：Q3, Q4, Q10
  - T/F（思考/情感）：Q5, Q6, Q11
  - J/P（判断/感知）：Q7, Q8, Q12
- 16种完整的猫咪人格类型结果，每种包含：
  - 独特称号（如"冷面霸总猫"、"梦游诗人猫"）
  - 3-5个性格标签
  - 100-140字的详细描述
  - 分享标题

#### 页面功能
- **首页**：展示标题、副标题和"开始测猫"按钮
- **测试页**：
  - 进度条显示（1/12 → 12/12）
  - 单选题目和两个选项
  - 点击选项自动进入下一题（300ms延迟反馈）
  - 最后一题显示"查看结果"按钮
- **结果页**：
  - 精美的猫咪3D渲染图片展示
  - 人格类型标识和称号
  - 性格标签和描述
  - "再测一次"和"返回首页"按钮

#### 视觉设计
- 温馨的橙米色系配色方案：
  - 主色：#FFA87D（柔和橙）
  - 辅助色：#FFD4B8（奶油橙）
  - 背景：#FFF8F0（米白）
  - 文字：#4A4A4A（柔和深灰）
- 大圆角、柔和阴影的现代UI风格
- 16张高质量3D渲染猫咪图片（已优化至600×900，约73-92KB/张）

#### 技术实现
- 微信小程序原生框架（WXML + WXSS + JS）
- 全本地数据存储（wx.Storage），无需后端
- 计分算法：每个维度3题，得分≥2取该字母
- 页面跳转采用合理的导航策略，避免页面栈溢出

### 技术规格
- 基础库版本：3.15.0
- 图片总大小：约1.7MB
- 支持按需加载组件（lazyCodeLoading）
