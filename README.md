# 猫咪CBTI人格测试小程序

一个可爱温馨的微信小程序，通过12道关于猫咪行为的问题，测试出猫咪的MBTI人格类型。

## 项目特点

- 🐱 **3D渲染素材**：16种猫咪类型，每种都有精美的3D渲染图片
- 🎨 **温馨设计**：柔和的橙米色系，大圆角，柔和阴影
- 📱 **原生小程序**：使用微信小程序原生框架开发
- 💾 **全本地数据**：无需后端，所有数据本地存储
- ✨ **流畅体验**：自动进题，300ms延迟反馈

## 功能说明

### 首页 (Home)
- 展示标题和副标题
- 提示文字：请按猫在多数情况下的表现选择
- "开始测猫"按钮

### 测试页 (Quiz)
- 进度条显示（1/12 → 12/12）
- 题目和两个选项
- 点击选项自动进入下一题（延迟300ms）
- 最后一题显示"查看结果"按钮

### 结果页 (Result)
- 精美的猫咪3D渲染图片
- 人格类型标识（如INTJ）
- 称号（如"冷面霸总猫"）
- 3-5个标签
- 100-140字描述
- "再测一次"和"返回首页"按钮

## 核心功能

### 计分算法

- 12题分4个维度，每个维度3题
- E/I维度：Q1, Q2, Q9
- S/N维度：Q3, Q4, Q10
- T/F维度：Q5, Q6, Q11
- J/P维度：Q7, Q8, Q12
- 每个维度得分≥2即取该字母
- 最终拼成4字母类型（如ENTJ）

### 数据流转

```
首页 → 清空存储
  ↓
测试页 → 记录分数到本地存储
  ↓
结果页 → 读取分数 → 计算类型 → 展示结果
```

### 页面跳转

- 首页 → 测试页：`wx.navigateTo`（可返回）
- 测试页 → 结果页：`wx.redirectTo`（不可返回）
- 结果页 → 首页/测试页：`wx.redirectTo`（重置流程）
